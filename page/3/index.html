Apache Shiro框架

本文字数： 0 阅读时长 ≈ 1 分钟

概述

    Apache Shiro是 Java 的一个强大易用的安全（权限）框架，提供了认证、授权、加密、会话管理、与Web 集成、缓存等功能，可为任何应用提供安全保障，能非常容易的开发出足够好的应用，其不仅可以用在JavaSE 环境，也可以用在 JavaEE 环境。Shiro为解决下列问题提供了保护应用的API：

  • 认证 - 用户身份识别，常被称为用户“登录”；
  • 授权 - 访问控制；
  • 密码加密 - 保护或隐藏数据防止被偷窥；
  • 会话管理 - 每用户相关的时间敏感的状态。
